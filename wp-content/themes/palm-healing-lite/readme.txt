== Description ==

Palm Healing Lite is a massage beauty salon and spa related website template. It can be used for health care, manicure, pedicure, health clubs, essential oils business, after-cure, facial, fat farm, hot spring, ayurveda, hairdresser girly, barber, therapist, treatment, depression, stress, nail salon, hair, Hydro cure, medical parlor, wellness center, aroma therapy, physiotherapy, reiki, meditation, yoga, fitness among other businesses. Coded using Elementor is compatible with a lot of popular plugins for shop, SEO, contact forms and others.
